Forecasting TSA Data with R

2024-09-20

If we want to create a forecast in R for the TSA Passenger data to predict passenger volumes in the future, we can use the tidyverts packages `fable`, `tsibble`, and `feasts`.

These packages apply tidy data principles to time series data and forecasting. These packages were coauthored by the forecasting great Rob Hyndman, and his fantastic textbook Forecasting: Principles and Practice (3rd Edition) uses these packages and workflows.
